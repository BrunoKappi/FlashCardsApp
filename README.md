# Flashcards.io 🚀

An advanced, premium study tool that allows users to create their own categories and cards, organize them, or play an automatically generated trivia game. 

Migrated to a modern frontend architecture using **Vite, React, TypeScript, Redux, and Tailwind CSS v4**, featuring a premium design inspired by **shadcn/ui**, full **Light/Dark Mode** support, and **multilingual translation** (English, Portuguese, Spanish).

## Demo
🚀 [Flashcards.io Live App](https://flashcards.bkappi.com/)

## Core Stack
- **Bundler & Tooling**: Vite
- **Language**: TypeScript
- **UI & Styling**: Tailwind CSS v4 (inspired by shadcn/ui palette & components)
- **State Management**: Redux
- **Routing**: React Router DOM
- **Interactions**: React Beautiful DnD

## Key Features
- **Light & Dark Theme**: Sleek background adapting to system preference or manual toggle.
- **Multilingual Support (i18n)**: Fully translated interfaces in English, Portuguese, and Spanish.
- **Trivia Game Creator**: Pull random questions automatically by category and amount via OpenTDB API.
- **Card Creator & Manager**:
  - Add, edit, and delete custom categories.
  - Create cards of different types: **Text Answer** or **Multiple Choice**.
  - Edit and delete cards inside custom categories.
  - Drag and drop cards to organize.
- **Interactive Gameplay**: Flip cards to reveal answers, choose multiple choice options, check scores, and review rounds with visual metrics charts.
- **LGPD & GDPR Compliance**:
  - **Cookie Consent Banner**: Allows users to manage consent and cookie choices local to their device.
  - **Standalone Legal Pages**: Fully dedicated pages for Terms of Use, Privacy Policy, Cookie Policy, and Disclaimers.
  - **Right to Erasure (LGPD)**: Actionable "Delete My Data" tool in the Privacy Policy to completely wipe local storage settings and cookies.
- **Developer Section**: Dedicated "About the Developer" profile showcasing background, portfolio, blog, and professional contacts.
- **Info Dropdown Popover**: Clean navigation popover inside the header offering quick access to legal documents and developer details.
